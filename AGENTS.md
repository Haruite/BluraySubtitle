# Repository Modification Instructions / 仓库修改指令

Before modifying this repository, read and follow both language versions of the mandatory standards:

- [Code Modification Standards](docs/development/code-standards.md)
- [代码修改规范](docs/development/code-standards.zh-Hans.md)

The two versions describe the same rules and must be updated together whenever the author establishes a new rule.

After completing a refactoring or major change, update both history files in the same change. Ordinary changes do not require a history entry:

- [Refactoring History](docs/refactoring/refactoring-history.md)
- [重构历史](docs/refactoring/refactoring-history.zh-Hans.md)

修改本仓库前，必须阅读并遵守以上中英文代码修改规范。作者确定新规则时必须同步更新两个版本；完成重构或重大改动时必须在同一次修改中同步更新中英文重构历史，一般修改不需要增加历史记录。
